Provides a Search terms cloud widget in the front of your website.

== Description ==

Search terms cloud is a fully localized and costumizable widget for your website, it displays the most used terms in your website.


== Installation ==

1. You can either download the plugin from Wordpress.org directory or upload files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Settings > Search terms cloud" page to costumize the plugin.

== Frequently Asked Questions ==

= Is the plugin fully localized? =

Search terms cloud plugin is fully localized to fit your language needs.

= How can i costumize the plugin css styles? =

The plugin use a classname "search_terms_widget" that contain a serie of "a" and "span" tags that you can style within your css file.

= I have some settings that i can't change on the plugin? =

If you have any requests for some settings that you want to change you contact me and i will try to include it on the next release.
